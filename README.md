# NTM Translations Website

Website for **NTM Translations**, a Chennai-based translation and language services provider.

## Pages

- **Home**: Hero, services overview, why choose us, languages
- **About**: Company story, values, process
- **Services**: Document translation, localization, subtitling, interpretation, proofreading
- **Contact**: Email, location, quote request form
- **FAQ**: Common questions

## Preview Locally

Open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
